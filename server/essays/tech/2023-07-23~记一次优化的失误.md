# 记一次优化的失误
*作者：PAPEREE｜日期：2023-07-23*

我在B站刷着视频，发现了一个叫*Dism++*的系统优化软件，于是下来逝世。

在*系统优化*的项目里，我仿佛发现了新大陆，抱着*这些都只是小改动*、*随便改不会有问题*的心态，我几乎是把每个选项勾来勾去，看看效果后再决定改不改。

结果，不知道摁到哪个选项，直接一键清空了我桌面的全部图标图片，只剩下文件名能点击。包括文件夹侧栏图标、文件夹中的文件图标，也全都消失不见。

于是重启，并没有解决这个问题。

我慌了。少了图标，各类型文件和文件夹傻傻分不清，再者美观是个大问题。

我在bing上疯狂搜索寻找教程，事实证明几乎没用。比如说删掉隐藏文件*IconCache.db*再重启资源管理器、输入`DISM /Online /Cleanup-Image /CheckHealth`重建缓存什么的，全被我的电脑免疫掉。甚至还有*计算机重置*这种一劳永逸的方法，亏我放到了*若是真走投无路*的后头。

所幸我在[这篇文章](https://qastack.cn/superuser/1158007/icons-disappeared-from-my-desktop-in-windows-10-only-the-names-remain)中最后一条评论里找到了解决方法，如下：

用*Win+R*输入*regedit*打开注册表，再进入`HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Explorer`，将*ShellIcons*重命名或删除掉。为了保险起见，我选择在源文件后面加上*_*下划线。

至此，还真解决了这个问题。

好，说实话，以后再也不敢随便搞优化了，至少得先权衡下它是否值得优化。
