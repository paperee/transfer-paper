# 为何修改系统时间
**作者：PAPEREE｜灵感：MelonCmd｜日期：2023-10-22**

（本文纯属虚构，与现实中/网络上的一切人、事、组织无关）

溺死在了时间里，被推出梦境。 ——cmd

或许你听说过互联网博物馆。那是个会在不定期记录下任何网页形态的网站。
然而，时间点外无迹的网页形态，或者仅在刹那间存在，亦或是存在恰好错开前后两个时间点。

那天，网友cmd郑重其事地告诉我，唯有修改电脑系统时间可以破解空间的限制。
虽说现实中不会有任何变化，但只要使用##浏览器接收互联网电波，网页便是那个时间点存在的形态。

当然，更有意思的是，在那个时间点中，互联网上发生的一切都会重现。

是不是听起来十分玄乎？
的确，闻所未闻，更像是梦见。

cmd可就不乐意了，当即往纸床上扔了一个安装包，说这是##浏览器。
还不忘吐槽：这服务器用什么破宽带——那么慢。

我不顾火绒安全软件三番五次提示报毒，便打开运行。
这下##浏览器是安上去了，却捆绑安装了365 Soft全家桶。
包括365计时器、365锁屏、365桌宠、365 IM……等等。

…￥……￥%￥&*…&#……

在网络上对cmd进行无止境的谩骂后，我彻底忘了这回事。

那天，我准备迁移我的硬盘。
进度条起初爬得很快，突然在中间卡住。
我纳闷着检查后，发现有近万个以KB计数的txt碎片文件，储存在某个打开时间为13个月前的文件夹里。

那是曾经一个叫XC的网站留下的数据。
包括大部分机器人的源代码、大多数日子的聊天记录……等等。

那时，XC上的用户们是来自国内各地的网友，几乎无所不谈。
后来由于友谊饼干的掰断而终结，大家分道扬镳。

我记不得有多少次尝试访问老网址，502 Bad Gateway 打不开；
记不得有多少次翻遍互联网博物馆，404 Not Found 找不到；
记不得有多少次决定联系以前的网友，403 Forbidden 拒绝请求。

这么回忆着的我有些伤感，手抖着修改了系统时间，打开##浏览器——
啥##浏览器，不就是个套着Firefox皮、用着80.0.0版本Google老祖宗内核的缝合怪吗？！

打了个喷嚏，心中泛起一阵恶寒，还以为cmd让网友ZHR偷偷放rickroll。

我凭着记忆输入XC网址——原本停运的XC真的还活着。
黑中泛黄的背景，微软雅黑的字体，时宽时挤的间距，乱七八糟的侧栏。

```
在线用户：ZhangModHelper，_，azhe，OIer酱，mouse，挽风，10555，MelonFish，Shabby_Paper

Shabby_Paper
我不是ee 我是一张傻逼纸片
```

wtf，改错时间了。

我一见这个如同sb般的昵称，就连忙切了系统时间。
