# 失败的恶作剧
**作者：PAPEREE｜日期：2023-09-23**

事情是这样的，上周五下午操场上举办社团招新。

我~~作为科技社软件部部长，不帮忙摆摊，反而偷偷溜去~~与高一的好朋友18和02，在食堂买了猪排堡和柠檬茶，兴奋地走在场地上闲逛。

活像几个没见过世面的高一新生。

其他社团都不怎么瞩目，唯独舞蹈社和声乐社，永远都是围观人数最多的。

从5点玩到6点，回去之时，我猛地想到：既然每一个社团都有自家的海报，我是否可以做一张《***不存在的社团***》的海报，再像普通社团那样在高一每个班上做宣传？

没错。永远不会放弃……高一新生。

我还真开始行动。周六一回家，我试着问了朋友中最好（四声）玩的CJX。想不到，他激动地表示，自己早就想这么干。于是，我们的诈骗计划一拍即合。

首先是海报。CJX找了背景，我在上面添了文字。

CJX：*卧槽牛逼！以假乱真了都！*

仅用一个下午，我们便将事前工作搞好了。至于打印，CJX靠谱地接下了工作。

时间来到周三中午，我按约定时间到达软件室，想不到候在这里的不是CJX，而是LZM。我心里咯噔一下，懊悔道居然忘记了中午LZM也在。所幸他似乎不知此事。

一会儿，CJX蹦出来了，手里握着一捆A4纸，跟在他背后的……居然还有CJC！

CJC刚进门就迫不及待问道：*所以到底是什么事、软件社又是什么！*

LZM听完一脸懵逼，看看我们三人不知在说什么。

我还以为是CJX将事情败露了。

不料CJX指着CJC：*他妈的我刚打开书包，这家伙跑过来看我翻书包！看到了海报！*

CJC：*啊？什么什么，什么海报？！所以软件社到底是什么！*

\&…#…%%#%………%&……

此处省略一大段解释。

大致意思是，我联合CJX打算坑Python小分队其他成员及高一新生（包括在场的CJC和LZM）一把。我们打算往软件室门上贴海报，还打算趁午休高一楼人少，给高一的每个班都发一张海报。

CJC和LZM都很震惊。

不过这两人也加入了犯罪行列，~~自愿~~被迫成了从犯。

看来CJX准备充分，他掏出了一卷双面胶，给软件室门的前后面都贴上了海报。

接下来是愉快的发~~传单~~海报活动。20个班，4个人，每人拿5张，刚刚好。

CJC刚接过海报，表示他想发三楼的班，话音未落就往楼上狂奔。大家见他跑得如此之快，一个一个都很无奈。

我才拿到海报走到二楼某个班门口，见里面还有不少学生，便开始慌了，又往回走。

CJX：*去吧去吧，都到这一步了！*

虽说是硬着头皮进去，但好歹开了个头后再也不紧张。有的班讲台上围着一群学生，我进去丢一句“给你们看的”。遂迎着他们不解的神情，甩一张海报到讲台后匆忙逃走。

发完后，CJX发现多出一张。
