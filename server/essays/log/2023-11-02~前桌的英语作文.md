# 前桌的英语作文
## 关于文章
事情是这样的，我的好前桌——下文皆称其为抹布，今天突然让我去找他放在某处的一篇英语作文。

我从他的微信翻到QQ，毫无所获，失落极了。

下午洗完澡，猛地想起一个提示——作文放在我一个月前浏览过的地方。

我思考着，有没有一个关于我、且其他用户也可以留言的平台——博客（

但我家那足以用KB/s计数的网络，压根打不开坐落在美国的博客，于是作罢。

在我的再三追问下，抹布给我发了一张图：

![](https://fs-im-kefu.7moor-fs1.com/29397395/4d2c3f00-7d4c-11e5-af15-41bf63ae4ea0/1707307591526/%E6%8A%B9%E5%B8%831.webp)

啊？？！

我赶紧去翻博客——

将所有文章的评论区翻了个底朝天，依旧没有看到所谓的英语作文。

直到抹布睡前，我还在刨根问底。

过一会儿，他回复道：`“明明按了POST”`。

完蛋——为了防止刷屏，评论区被我加上了严格的字数限制。

真对不起抹布啊啊啊。

结果，抹布看起来十分得意，表示他还留了一手——

![](https://fs-im-kefu.7moor-fs1.com/29397395/4d2c3f00-7d4c-11e5-af15-41bf63ae4ea0/1707307591788/%E6%8A%B9%E5%B8%832.webp)

聪明的抹布君！
